# Video Editor

Editor de video de escritorio construido con Electron y FFmpeg.

## Características
- Importar videos e imágenes
- Timeline con múltiples clips
- Recorte, velocidad, color y filtros
- Exportar a MP4

## Requisitos
- Node.js
- FFmpeg instalado en el sistema

## Instalación
git clone https://github.com/K-A-Y-R-U/video-editor.git
cd video-editor
npm install
npm start

## Tecnologías
- Electron
- FFmpeg / fluent-ffmpeg
